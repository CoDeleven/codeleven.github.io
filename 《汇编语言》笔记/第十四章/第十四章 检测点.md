## 检测点14.1 

### 编程，读取CMOS RAM的2号单元内容。
```aeemble
assume cs:codesg

codesg segment

start:
    mov al, 2
    ; 向70H端口 发送锁定 2号单元的信号
    out 70H, al
    ; 执行完该条指令后，al里的数据为当前时间 的 分钟
    in al, 71H 

    mov ax, 4c00H
    int 21h
codesg ends
end start
```
